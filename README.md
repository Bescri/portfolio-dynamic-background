# Usage

## Linux

          sudo apt install git
          sudo apt-get update
          
          mkdir portfolio
          cd portfolio
          git clone https://github.com/Bescri/portfolio-dynamic-background
          cd portfolio-dynamic-background/
          
          
          Optional: npm start
